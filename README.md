ภาพโดยรวมของโปรเจค / Example for preview.
Details From LOHA's : Asp.net8 core Web API + AdminLTE for prob  

Backend (ฝั่งเซิร์ฟเวอร์)
ASP.NET Core MVC (.NET 8.0)	Framework หลักสำหรับสร้างเว็บแอป
Entity Framework Core	ORM สำหรับจัดการฐานข้อมูล
ASP.NET Core Identity	ระบบ Login/Authentication
SQL Server	ฐานข้อมูล

Frontend (ฝั่งผู้ใช้)
AdminLTE	Template Dashboard (ทำให้หน้าตาเว็บดูสวยและพร้อมใช้)
Chart.js	สร้างกราฟแสดงข้อมูลพนักงาน
Bootstrap 4	Framework CSS สำหรับ Responsive Design
jQuery	จัดการ JavaScript ได้ง่ายขึ้น
Font Awesome	Icons สวยๆ
FullCalendar	ปฏิทินในหน้า Dashboard
Custom CSS (premium-theme.css)	ธีมสีดำ-แดงที่ผมสร้างเพิ่ม

พิเศษ
CSS Animations (กองไฟในหน้า Login) 
Razor View Engine - สำหรับรวม C# + HTML
